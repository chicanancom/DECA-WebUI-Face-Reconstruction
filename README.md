# DECA Web UI - 3D Face Reconstruction

"This is a personalized version of DECA with a Web UI for research purposes. All rights belong to the original authors."

Web-based playground for [DECA](https://github.com/YadiraF/DECA)
Powered by Three.js

## Upload images or videos directly in your browser.

## How to run

1. Visit [deca site](https://github.com/YadiraF/DECA) to know how to download the model and run it.
2. run `pip install -r requirements.txt`
3. run `python web/backend/main.py`
4. this web ui will run on [http://localhost:8000](http://localhost:8000)
